This is a web randomizer allowing you to choose your wolf as the main subject. I chose to use a wolf because it's my spirit animal and I admire them. This randomizer is for you to choose which wolf is for you from emotions, rank, and characteristic trait. It would be your wolf personality.

The program software it is using is Javascript, HTML, and CSS all done on Atom.io and GitHub.
